Description

Componente de la página principal del administrador.

Este componente representa la página principal para los usuarios con privilegios de administrador. En su plantilla, se integra un componente de barra lateral (sidebar) que puede contener enlaces de navegación y otras herramientas para el administrador.

Example :
<app-admin-homepage></app-admin-homepage>
